## .NET Commands

    dotnet new -i Microsoft.MobileBlazorBindings.Templates::0.5.50-preview
    dotnet new blazorhybrid -o MyApp
    dotnet new classlib --name MyApp.Shared
    dotnet new blazorwasm --name MyApp.Wasm
